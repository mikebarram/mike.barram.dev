date: "2010-11-25"

---

Don’t name a field ID, id or Id

If you ask NetTiers to generate an admin website, then it will show warnings when compiled but these will result in errors when you try to run the site.

Even if you don’t want an admin website to start with, you may want one later, so just don’t name any fields ID
