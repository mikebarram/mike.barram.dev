Yet more from the good people of Twitter [http://twitter.github.io/hogan.js/](http://twitter.github.io/hogan.js/). This is  a compiler for [\{\{mustache\}\}](http://mustache.github.io/), which is a templating language that can convert JSON to HTML in a similar way to Razor converting entities to HTML in ASP.Net. Useful if you want to say generate whole data grids on the client. If your server is getting JSON from elsewhere and you want to template it into HTML, you can use mustache templates on the server (rather than client) using [https://github.com/jdiamond/Nustache](https://github.com/jdiamond/Nustache)
